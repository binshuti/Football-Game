Overview

Mini Soccer Game is a 2D arcade-style football experience built in Python using Pygame. The player controls a striker (the ball) from a top-down perspective, weaving between moving defenders to score goals and progress through increasingly difficult levels.

Under the hood, the game uses a clean Model‚ÄìView‚ÄìController structure and includes features like smooth ball physics, animated defenders, lives, and a persistent high score saved to disk using JSON.

Key Features

  • Multiple levels with increasing difficulty and more defenders.
  • Smooth dribbling with inertia, friction, and ‚Äúescape assist‚Äù when defenders get close.
  • Defenders move horizontally and vertically in dynamic patterns with simple animation.
  • Crowd ambience, goal and tackle sound effects (WAV audio via pygame.mixer).
  • HUD with score, lives, and a high score stored in highscore.json.
  • Start menu and replayable game-over flow (no need to restart Python each time).

How to Run & Play

1. Install dependencies

pip install -r requirements.txt

2. Launch the game

python main.py

3. Controls

  • Arrow keys — move and dribble the ball.
  • Score by reaching the goal at the top of the field.
  • Avoid defenders — collisions cost you a life.
  • On game over, press R to play again or Q / ESC to quit.
